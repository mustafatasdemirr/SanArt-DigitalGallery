**SanArt**, edebiyat ve sanat tutkunları için tasarlanmış, gerçek zamanlı veri akışına sahip interaktif bir dijital anı duvarıdır. Van Gogh'un "Yıldızlı Gece" eserinden ilham alan temasıyla, kullanıcıların şiir ve notlarını bir sanat galerisi atmosferinde paylaşmasını sağlar.

## ✨ Öne Çıkan Özellikler

* **Real-time Veri Akışı:** Firebase Firestore ile sayfayı yenilemeden anlık güncelleme.
* **Sanatsal Arayüz:** Glassmorphism (cam efekti) ve akıcı geçişlerle zenginleştirilmiş kullanıcı deneyimi.
* **İnteraktif Öğeler:** Edebi sembollerin (tüy kalem, kitap) uçuştuğu dinamik arka plan ve müzik çalar entegrasyonu.
* **Duyarlı Tasarım (Responsive):** Mobil ve masaüstü cihazlarla tam uyumlu yapı.
* **Görsel Optimizasyon:** Kullanıcıların yüklediği görsellerin tarayıcı tarafında Base64 formatına çevrilerek optimize edilmesi.

## 🛠️ Kullanılan Teknolojiler

* **Frontend:** HTML5, CSS3 (Bootstrap 5, Glassmorphism), Vanilla JavaScript (ES6+ Modules).
* **Backend:** Firebase Cloud Firestore (NoSQL Veritabanı).
* **İkonlar:** FontAwesome.
